Fully Cross Platform

TrunxIO CLI wallet is a dependable and user-friendly EOSIO wallet you can use on ANY OS or architechture. No longer do you have to fire up your VMs to manage yet another wallet.

Highly Documented

Every command in the TrunxIO CLI wallet is fully documented including information about flags, args, default values, and more.

Powerful yet User-Friendly

TrunxIO CLI wallet is a fully featured wallet infrastructure solution that is clean, intuitive, and able to be imported/used in any JavaScript script/module.

Steps for Getting Started

Install TrunxIO CLI Wallet
JS binary/module

After installing TrunxIO CLI wallet binary, you will be able to use the trunxcli executable directly on your local machine to interact with the backend wallet database. Developers also have the ability to import the TrunxIO CLI wallet module in their own apps/scripts as well as create their own plugins for TrunxIO CLI wallet.

  • Highly Documented
  • Tech Support Available
  • Very User Friendly
  • Fully Open Source

Start the Backend
Database Service

The backend database service is the brains of the TrunxIO CLI Wallet. It is a long-running process responsible for the database management, administration, and security. For developers, TrunxIO CLI wallet can be imported as a module to communicate directly with the backend database service from other apps/scripts.

  • Run Attached for Debugging
  • Managed Databases Coming Soon
  • Reliable and Robust
  • Fully Open Source

Create Your First
TrunxIO CLI Wallet

With the backend service running, now it is time to create your wallet and get ready to have the power of EOSIO at your fingertips anywhere you go. Simply call the `db:unlock` command with the password you wish to encrypt your new database with. Be sure to always double check your passwords, backup your keys/databases, and test your login information before storing sensitive information inside.

  • AES-256-GCM Encryption
  • Configurable Auto-timeout Lock
  • Easily Manage Multiple Databases
  • Backend Database Audit Log

Import Your EOSIO
Keys and Accounts

Once your new TrunxIO CLI wallet database is created, finish up by adding your EOSIO chains, keys, and accounts. Alternatively, you may choose to import an existing wallet backup from any of the following wallets: TrunxIO, Anchor, Scatter. Enjoy your newly gained freedom amongst the EOSIO ecosystem (more chain integrations coming soon!).

  • Derive Keys from Mnemonic
  • Import Backups from Scatter/Anchor
  • Manually Import Keys
